Chess Rules

Basic Rules

  • White moves first.
  • One piece is moved per turn.
  • If a piece moves into a space with another piece, that piece is captured and removed from the board
  • Can not capture pieces of the same color.
  • Can not move over any piece. (excluding Knights: See movement)
  • If a player has no valid moves and is not in checkmate the game is considered a stalemate

Movement

  • Rook: Can move any number of spaces horizontally or vertically. Can not move diagonally. (2 each side)

  • Knight: Can move in L shapes - Out 2 spaces horizontally or vertically followed by 1 space vertically or horizontally (opposite of the 2 spaces movement). Can move either direction on the horizontal or vertical for both. Only piece that can move over other pieces. Captures opposing piece in the space the knight ends its turn in. (2 each side)

  • Bishop: Can move any number of spaces diagonally. Can not move horizontally or vertically. (2 each side)

  • Queen: Can move any number of spaces in any one direction (horizontally, vertically, diagonally). (1 each side)

  • King: Can move 1 space in any one direction (horizontally, vertically, diagonally). (1 each side)

  • Pawn: Can move 2 spaces forward on a piece's first move. Otherwise, it can only move 1 space. It can not capture pieces in front of it. It can only capture pieces Diagonally. (8 each side)

Objective

  • The game ends when a king () is taken or is in danger of being taken (check) and there are no valid moves to remove the danger (mate).
